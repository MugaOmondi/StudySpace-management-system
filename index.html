STUDY SPACE

This is a prototype for a studyspace management system. The system design for the project will implement fullstack tools such as Javascript for user experience, CSS and Html for content management, MYSQL and php which will handle backend processess. API integrations will help handle user email address authentication and subscriptions will harness the capabilities delivered by Daraja 2.0 for payments abd checkouts using M-pesa API
